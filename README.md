# AWS-IAM-Cloud-Security-Project
I completed this project on cloud security controls in Amazon Web Services (AWS), focusing on Identity and Access Management (IAM). The goal was to create a least‑privilege policy, attach it to a user group, and verify that the policy correctly restricts actions on Amazon EC2 instance

---

### IAM GROUP CREATED
- Abatech-processing-documentation
  - Administrativeaccess policy attached to it

### IAM USER CREATED
- Abatech-processing-documentation-jude

---
### POLICY 
- Policy created and attached to user - _Abatech-processing-documentation-jude_

### CLOUD TRAIL   
- Cloud trail with the name _management event_ created

### EC2
- An instance with the name _Documentation server_ was created using Windows AMI
- Instance was later stopped and terminated

### S3 BUCKET
- S3 bucket with name _Documentation bucket_ was created
- Document was uploaded and later deleted
